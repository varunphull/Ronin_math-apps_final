# Ronin's Maths Interactive Tool

Cambridge IGCSE Year 7/8 — Interactive maths explorer

## Files in this package

```
index.html          ← Main app (rename from ronins-maths-tool-v2.html)
manifest.json       ← PWA manifest (makes it installable)
service-worker.js   ← Offline caching
icon-192.png        ← App icon (Android / browser)
icon-512.png        ← App icon (high-res)
```

---

## How to Deploy (GitHub Pages — Free)

1. Go to github.com → Create new repository → name it `maths-tool`
2. Upload ALL 5 files (rename the HTML to `index.html`)
3. Go to Settings → Pages → Source: main branch → Save
4. Your app will be live at: `https://YOUR-USERNAME.github.io/maths-tool`

---

## How to Install on iPhone

1. Open the URL in **Safari** (must be Safari, not Chrome)
2. Tap the **Share** button (box with arrow pointing up)
3. Tap **"Add to Home Screen"**
4. Tap **Add** — icon appears on home screen
5. Open it — runs fullscreen like a native app ✅

---

## How to Install on Android

1. Open the URL in **Chrome**
2. Chrome will show a banner: **"Add to Home Screen"** — tap it
   OR: tap the 3-dot menu → "Add to Home Screen" / "Install App"
3. Icon appears on home screen ✅

---

## How to Install on Desktop (Windows/Mac)

1. Open in Chrome or Edge
2. Look for the install icon (⊕) in the address bar
3. Click it → Install
4. App opens in its own window, works offline ✅

---

## Updating the App

When you update `index.html`:
1. Upload the new file to GitHub
2. In the app, an **"Update Available"** banner will appear
3. Click **"Update Now"** — refreshes automatically

Or hard refresh: `Ctrl+Shift+R` / `Cmd+Shift+R`

---

## Offline Mode

Once visited once in a browser, the app works fully offline.
- All topics, sliders, drag points work without internet
- Only Google Fonts may fall back to system font if never cached

---

## Topics Covered

**Geometry & Mensuration**
Gradient, Midpoint, Bearings, Transformations, Area, Parallelogram & Trapezium,
3D Surface Area (Triangular Prism, Pyramid, Cuboid, Cylinder, Cone, Sphere, Penta/Hex Prism), Circles, Unit Conversion

**Algebra & Number**
Linear Equations, Intersecting Graphs, Nth Term, Percentages, Ratio & Proportion

**Statistics & Data**
Bar Charts (single + dual), Pie Charts, Time Series, Stem & Leaf, Frequency Diagrams

**Probability**
Single Events, Complementary, Two Dice
